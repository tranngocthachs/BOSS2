All student pages

All student pages have the same basic layout, which is described here.

Test Submission page

The Test Solution page allows you to select tests to run on a solution that you have submitted for an assignment.

Testing a submission

The main area of the page displays the following details about the submission that is to be tested:

  • The Module that the submission was submitted to. This can be clicked to take you to the Module Details page for that module.
  • The Assignment that the submission was submitted to. This can be clicked to take you tot he Assignment Details page for that Assignment.
  • The time that the submission was performed.

Under this there are check boxes to allow you to select which tests you wish to run on your submission. If there are no tests, the checkboxes will be replaced with a message saying 'There are no tests for this assignment'.

To test your submission, check the boxes to select the tests that you wish to run and click 'Test Submission'. If you do not wish to test your submission, you can navigate away from the page using the breadcrumbs, menu, or browser back button.
