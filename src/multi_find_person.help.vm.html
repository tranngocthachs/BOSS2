Find person page

The find person page allows you to query the database to find an individual.

Finding a person

The main part of the page contains a form for finding a person. You may specify the following criteria in the table to find an individual:

  • 'Name' criterion, which if specified will search inside the person's chosen name.
  • 'Unique ID' criterion, which if specified will search inside the person's unique identifier (aka. username).

Either field can be blank. Each will match a portion, for example, 'Doe' will match 'John Doe' or 'Doe John' or 'John Doeson'. To perform the search, click the 'Perform Search' button. If you do not wish to perform a search, navigate away from the page using the breadcrumbs, menu, or browser back button.
