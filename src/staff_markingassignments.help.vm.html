All staff pages

All staff pages have the same basic layout, which is described here.

Staff marking assignments page

The marking assignments page shows you all of the marking assignments for a given assignment. Marking assignments bind a student to a marker.

Creating a marking assignments

To create a new marking assignments, click 'Create new marking assignments...' under the Actions menu on the left hand side of the page.

Auto-create marking assignments

Creating marking assignments for every single marker and student can be a chore. BOSS2 allows you to automatically create marking assignments. To do this, click 'Auto-assign markers...' in the Actions menu on the left hand side of the page.

Marking assignments table

The main part of page contains a table describing the marking assignments in the current assignment. The table contains the following information for each marking assignment, one marking assignment to a row:

  • 'Student' column, which displays the unique identifier (aka. username) of the student and their name below it. You may click on the name to email the student.
  • 'Marker' column, which displays the unique identifier (aka. username) of the marker and their name below it. You may click on the name to email the marker.
  • 'Blind marking?' column, which displays whether the marker will be able to see the name of the student that they are marking.
  • 'Moderating?' column, which displays whether the marker will be able to see marks created for the student by other markers.
  • 'Actions' column, which an 'Edit/Delete' button. Click this button to edit or delete the marking assignment.
You can click the headers of the columns to sort the tables. Clicking the same header twice will reverse the direction. The following headers may be used for sorting:
  • 'Student', which will sort by the student's unique identifier (aka. username).
  • 'Marker', which will sort by the marker's unique identifier (aka. username.)
  • 'Blind marking', which will group marking assignments where the marker is not able to see the name of the student they are marking.
  • 'Moderating?', which will group marking assignments where the marker is able to see marks created for the student by other markers.
