All staff pages

All staff pages have the same basic layout, which is described here.

Staff results page

The results page shows you all of the published results for a given assignment. Results are the final marks that can be mailed out to students and are immutable.

Mailing results

To mail results out to the students, click 'Email results' under the Actions menu on the left hand side of the page.

Results table

The main part of page contains a table describing the results for the current assignment. The table contains the following information for each result, one result to a row:

  • 'Student' column, which displays the unique identifier (aka. username) of the student, with the chosen name below it.
  • 'Timestamp' column, which displays the time the result was published.
  • 'Result' column, which displays the percentage mark the student obtained.
  • 'Status' column, which displays whether the result was created from complete or incomplete marks.
