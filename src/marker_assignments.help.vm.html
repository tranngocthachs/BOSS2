All marker pages

All marker pages have the same basic layout, which is described here.

Assignments page

The assignments page shows all of the assignments that you have students to mark for and provides you access to the students that you are assigned to mark.

'Show..' menu

On the left hand of the page, under the navigation menu, is a menu that allows you to control which assignments are shown on this page. The items in the menu are:

  • 'Open assignments', which will display assignments that are open and which you have students to mark for.
  • 'Closed assignments', which will display assignments that have closed and which you have students to mark for.
  • 'All assignments', which will display all assignments that you have a student to mark for.

Assignments table

The main part of the page contains a table displaying information on the assignments that you are assigned to mark. The table provides the following information:

  • 'Assignment' column, which will display the name of the assignment and the name of the parent module below it.
  • 'Students available to mark' column, which will display the number of students for the assignment in the 'Assignment' column that you are assigned to mark.
  • 'Actions' column, which will contain a 'Mark/moderate this assignment' button. Clicking this button will take you to a list of students to mark for the assignment in the 'Assignment' column.

The table may be sorted by clicking on the column headers. Clicking twice on the same header will reverse the sorting order. You can click on the following headers to sort the table:

  • 'Students available to mark', which will sort by the number of students an assignment has available to mark.
