All admin pages

All admin pages have the same basic layout, which is described here.

Edit Person page

This page allows you to edit, delete or create a person.

Creating/editing the person

A person has the following values that must be filled in:

  • 'Unique identifier (aka. Username)': a small textual identifier for the person. No two people may have the same unique identifier. This doubles up as the username for the user that is used to log in by.
  • 'Chosen name': the real name of the user, for example 'John H. Smith'. Names do not need to be unique.
  • 'Email address': the email adddress of the user, used for all communication with the user.
  • 'Password': the password of the user. If you are editing the peson's details you can leave this blank to leave the person's password unchanged.
  • 'Administrator': whether the individual has access to the administration section or not.

Deleting the person

If you are not creating a person, under the values for the person there will be a checkbox marked 'Delete this person'. To delete the person rather than edit it, select this checkbox. If you are creating a person this checkbox will not be visible.

NOTE: If the person has submissions or other data in the BOSS2 system, you will not be able to remove the person until the data is removed.

Confirming

If you are editing or deleting a person there will be a button labeled 'Perform Edit/Delete'. If you are creating a person the button will be labeled 'Perform Create'. Once you are sure of the values you have selected, click this button to commit the details to the BOSS2 database. If you do not wish to create, edit or delete a person navigate away from the page using the navigation menu, breadcrumbs or browser back button.
