All admin pages

All admin pages have the same basic layout, which is described here.

Edit Module page

This page allows you to edit, delete or create a module.

Creating/editing the module

A module has the following values that must be filled in:

  • 'Unique identifier': a small textual identifier for the module. No two modules may have the same unique identifier.
  • 'Name': a custom name for the module. Names do not need to be unique and may be longer.
  • 'Requires registration': a flag stating whether students need to register to the module to be allowed access to it.

Deleting the module

If you are not creating a module, under the values for the module there will be a checkbox marked 'Delete this module'. To delete the module rather than edit it, select this checkbox. If you are creating a module this checkbox will not be visible.

NOTE: If the module has assignments you will not be able to delete the module until those assignments are deleted.

Confirming

If you are editing or deleting a module there will be a button labeled 'Perform Edit/Delete'. If you are creating a module the button will be labeled 'Perform Create'. Once you are sure of the values you have selected, click this button to commit the details to the BOSS2 database. If you do not wish to create, edit or delete a module navigate away from the page using the navigation menu, breadcrumbs or browser back button.
