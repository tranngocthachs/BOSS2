All admin pages

All admin pages have the same basic layout, which is described here.

Edit Model page

This page allows you to edit, delete or create a model.

Creating/editing the model

A model has the following values that must be filled in:

  • 'Unique identifier': a small textual identifier for the model. No two models may have the same unique identifier.
  • 'Name': a custom name for the model. Names do not need to be unique and may be longer.

Deleting the model

If you are not creating a model, under the values for the model there will be a checkbox marked 'Delete this model'. To delete the model rather than edit it, select this checkbox. If you are creating a model this checkbox will not be visible.

Confirming

If you are editing or deleting a model there will be a button labeled 'Perform Edit/Delete'. If you are creating a model the button will be labeled 'Perform Create'. Once you are sure of the values you have selected, click this button to commit the details to the BOSS2 database. If you do not wish to create, edit or delete a model navigate away from the page using the navigation menu, breadcrumbs or browser back button.

Note: if the model has modules, you will not be able to delete it until you remove the sub-modules.
