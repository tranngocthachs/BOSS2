All staff pages have the same basic layout, which is described here.

Edit Test parameters page

This page allows you to specify parameters for your test.

Editing test parameters

The page provides a list of parameters for you to edit. These will be specific to the test being edited.

Each parameter will have a name, and a description providing help on what you should specify for the parameter.

Confirming

Click the 'Update' button to confirm. It is recommended to always do this, even if you do not modify the test parameters.
