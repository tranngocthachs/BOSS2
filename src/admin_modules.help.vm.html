All admin pages

All admin pages have the same basic layout, which is described here.

Admin modules page

The modules page shows you all of the modules for a given model. A module is a collection of assignments, for example a subject on a course.

Creating a new module

To create a new module, click the 'Create new module' item in the 'Actions' menu on the left hand side of the page.

Modules table

The main part of page contains a table describing the modules in the current model. The table contains the following information for each module, one module to a row:
  • 'Module' column, which displays the unique identifier of the module and below it its name. You can click the identifier of the module to edit it.
  • 'Assignments' column, which displays the number of assignments under the module.
  • 'Administrators' column, which displays the number of administrators for the module.
  • 'Registration Required' column, which displays whether students must be registered to the module to access it.
  • 'Actions' column, which displays a button marked 'Edit/Delete Module' and 'Manage Administrators', described in the next section.
You can click the headers of the columns to sort the tables. Clicking the same header twice will reverse the direction. The following headers may be used for sorting:
  • 'Module', which will sort by the module's unique identifier.
  • 'Assignments', which will sort by the number of assignments under the module.
  • 'Administrators', which will sort by the number of administrators under the module.
  • 'Registration Required', which will modules by whether they require registration or not.

Actions

The actions for each module can be performed by clicking the button in the 'Actions' column for the module. These actions are:
  • 'Edit/Delete Module', which will allow you to edit the module's parameters or delete it.
  • 'Manage Administrators', which will allow you to add or remove administrators for the module.
