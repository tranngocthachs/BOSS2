Found person page

The found person page shows you the results of your search and allows you to add a person in the context of the search.

Results

The main part of the page contains a table of the found people. The table contains the following:

  • 'ID' column, which displays the unique identifier (aka. username) of the person.
  • 'Name' column, which displays the name of the person.
  • 'Email' column, which displays the e-mail address of the person. This may be clicked to compose an email to the individual.
  • 'Use?' column, which contains an 'Add..' button. Clicking this button will add the person to the context of the search, for example, if searching for a module administrator it will add the person as one of the given module's administrators.

If you do not wish to add an individual, navigate away from the page using the breadcrumbs, menu, or browser back button.
