All staff pages

All staff pages have the same basic layout, which is described here.

Staff marking categories page

The marking categories page shows you all of the marking categories for a given assignment. Marking categories are particular aspects of a solution that should be marked. Each has a weighting in the final mark, and for a result to be complete there should be a mark provided for each marking category.

Creating a marking category

To create a new marking category, click 'Create new marking category...' under the Actions menu on the left hand side of the page.

Marking categories table

The main part of page contains a table describing the marking categories in the current assignment. The table contains the following information for each marking category, one marking category to a row:

  • 'Name' column, which displays the name of the marking category.
  • 'Weighting' column, which displays the weighting for the marking category.
  • 'Actions' column, which an 'Edit/Delete Marking Category' button. Click this button to edit or delete the marking category.
You can click the headers of the columns to sort the tables. Clicking the same header twice will reverse the direction. The following headers may be used for sorting:
  • 'Weighting', which will sort by the marking category's weighting
