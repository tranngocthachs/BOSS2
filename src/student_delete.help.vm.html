All student pages

All student pages have the same basic layout, which is described here.

Delete Submission page

The Delete Submission page lets you confirm that you want a previous submission for an assignment to be deleted.

Deleting a submission

The main area of the page displays the following details about the submission that is to be deleted:

  • The Module that the submission was submitted to. This can be clicked to take you to the Module Details page for that module.
  • The Assignment that the submission was submitted to. This can be clicked to take you tot he Assignment Details page for that Assignment.
  • The time that the submission was performed.

Under this there is a check box to confirm that you wish to delete the submission, and a button marked 'Delete Submission'.

To delete your submission, check the box to confirm and click 'Delete Submission'. If you do not wish to delete your submission, you can navigate away from the page using the breadcrumbs, menu, or browser back button.

Note: if the module administrator has disabled deleting submissions the 'Delete Submission' button will not be present, and instead there will be a notice stating 'Sorry. The assignment administrator has disallowed submission deletion.'
