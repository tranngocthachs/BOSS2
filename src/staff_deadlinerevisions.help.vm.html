All staff pages

All staff pages have the same basic layout, which is described here.

Staff deadline revisions page

The deadline revisions page shows you all of the deadline revisions for a given assignment. Deadline revisions are a new deadline for a student, overriding the closing date of the assignment.

Creating a deadline revision

To create a new test, click 'Create new deadline revision...' under the Actions menu on the left hand side of the page.

Deadline revision table

The main part of page contains a table describing the deadline revision in the current assignment. The table contains the following information for each deadline revision, one deadline revision to a row:

  • 'Student' column, which displays the unique identifier of the student (aka. username). Below this is the chosen name of the student. You may click the chosen name to compose an email to the student.
  • 'New deadline' column, which displays the new deadline for the student.
  • 'Comment' column, which displays the reason for which the deadline was created.
  • 'Actions' column, which an 'Edit/Delete' button. Click this button to edit or delete the deadline revision.
You can click the headers of the columns to sort the tables. Clicking the same header twice will reverse the direction. The following headers may be used for sorting:
  • 'Student', which will sort by the student's identifier (aka. username)
  • 'New deadline', which will sort by the student's new deadline.
