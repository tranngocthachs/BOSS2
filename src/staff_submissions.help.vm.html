All staff pages

All staff pages have the same basic layout, which is described here.

Staff submissions page

The submissions page shows you all of the submissions for a given assignment. These submissions were made by students.

Submissions table

The main part of page contains a table describing the submissions in the current assignment. The table contains the following information for each submission, one submission to a row:

  • 'Name' column, which displays the unique identifier (aka. username) of the student that made the submission, with their chosen name underneath.
  • 'Submission Time' column, which displays the time the submission was received by the BOSS2 system.
  • 'To be marked' column, which displays whether the student wants the submission to be marked.
  • 'Actions' column, which displays a 'Download' button. Click this to download the submission as it was received by the BOSS2 system.
You can click the headers of the columns to sort the tables. Clicking the same header twice will reverse the direction. The following headers may be used for sorting:
  • 'Name', which will sort by the student's unique identifier (aka username).
  • 'Submission time', which will sort by the time the submission was received by the BOSS2 system.
