All admin pages

All admin pages have the same basic layout, which is described here.

Admin module administrators page

The module administrators page shows you all of the administrators for a given module. An administrator for a module is allowed access to the staff section to administrate the module.

Add a new administrator

To add an administrator, click the 'Add administrator' item in the 'Actions' menu on the left hand side of the page.

Administrators table

The main part of page contains a table describing the administrators in the current module. The table contains the following information for each administrator, one administrator to a row:
  • 'Person' column, which displays the identifier for the person and below it the person's chosen name.
  • 'Email' column, which displays the e-mail address of the administrator. You may click the address to compose an email to the administrator.
  • 'Actions' column, which displays a button marked 'Remove administrator'. Click this button to remove the administrator from the module.
You can click the headers of the columns to sort the tables. Clicking the same header twice will reverse the direction. The following headers may be used for sorting:
  • 'Person', which will sort by the administrator's unique identifier.
  • 'E-mail Address', which will sort by the administrator's email address.
