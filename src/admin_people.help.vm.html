All admin pages

All admin pages have the same basic layout, which is described here.

Admin people page

The people page lists all of the people in the database and allows you to add a new person, or edit/delete an existing person.

Creating a new person

To create a new person, click the 'Create new person' item in the 'Actions' menu on the left hand side of the page.

People table

The main part of page contains a table describing the people in the database. The table contains the following information for each person, one person to a row:
  • 'Person' column, which displays the identifier for the person and below it the person's chosen name. You can click the name of the person to edit their details.
  • 'Administrator?' column, which displays whether the person is an administrator or not.
  • 'Actions' column, which displays a button marked 'Edit/Delete Person', which you can click to edit or delete the person.
You can click the headers of the columns to sort the tables. Clicking the same header twice will reverse the direction. The following headers may be used for sorting:
  • 'Person', which will sort by the person's unique identifier.
  • 'Administrator?', which will group together administrators and non-administrators.
