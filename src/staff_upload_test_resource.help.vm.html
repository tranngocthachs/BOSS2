All staff pages have the same basic layout, which is described here.

Upload test resource page

This page allows you to upload an test resource. Test resources are available to the command executors for running tests and could contain libraries, programs to run, and/or reference data.

When the test is run, the test resource is unpacked into a directory which is passed to the command executor. For this reason the test resource must be a valid .zip or .jar archive (.jar archives are actually just .zip archives).

Uploading a test resource

To upload an test resource select the resource in the file choose under 'Resource file'.

Confirming

To confirm the upload, click the 'Perform Upload' button. This will overwrite any existing test resource that was uploaded. If you do not wish to upload a test resource, navigate away from the page using the menu, breadcrumbs or browser back button.
