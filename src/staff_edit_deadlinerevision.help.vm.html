All staff pages

All staff pages have the same basic layout, which is described here.

Edit Deadline Revision page

This page allows you to edit, delete or create a deadline revision. Deadline revisions are a new deadline for a student, overriding the closing date of the assignment.

Creating/editing the test

A test has the following values that must be filled in:

  • 'New deadline': the new date that the student must submit their solution by.
  • 'Student': the student that the deadline revision is for.
  • 'Reason': a comment explaining why the deadline revision was granted.

Deleting the deadline revision

If you are not creating a deadline revision, under the values for the module there will be a checkbox marked 'Delete this deadline revision'. To delete the deadline revision rather than edit it, select this checkbox. If you are creating a deadline revision this checkbox will not be visible.

Confirming

If you are editing or deleting a deadline revision there will be a button labeled 'Perform Edit/Delete'. If you are creating a deadline revision the button will be labeled 'Perform Create'. Once you are sure of the values you have selected, click this button to commit the details to the BOSS2 database. If you do not wish to create, edit or delete a deadline revision navigate away from the page using the navigation menu, breadcrumbs or browser back button.
