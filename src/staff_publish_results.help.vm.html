All staff pages

All staff pages have the same basic layout, which is described here.

Publish results page

The publish results page allows you to convert marks into the database into firm results, ready to be mailed out to students.

Publishing results

To publish the results for students, select the confirmation checkbox and click the 'Perform' button.

If you do not wish to publish the results, you can navigate away from the page using the breadcrumbs, menu, or browser back button.

NOTE: If students have not made submissions or marking is not complete for a particular student, these students will be displayed on the page. It is possible to publish results when no or partial marks are available, those missing marks will be assumed to be zero.
