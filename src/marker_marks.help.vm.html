All marker pages

All marker pages have the same basic layout, which is described here.

Marks page

The marks page displays marks created for a student on a particular assignment. If you are a normal marker, only your marks will be displayed. If you are a moderator, marks from all markers will be displayed (though you will only be able to edit your own marks).

Creating a new mark

Under the navigation menu is an 'Actions' menu. Clicking 'Create new mark...' in this menu will create a new mark and take you to the Edit Mark page for that mark.

Marks table

The main part of the page contains a table displaying information on the marks that you are available for the given student on the given module. The table provides the following information:

  • 'Marker' column, which will display the identifier and name of the marker that created the mark. Clicking the name will allow you to email the marker.
  • 'Marking category' column, which will display the name of the marking category that the mark is under.
  • 'Timestamp' column, which will display the time that the mark was last edited (or created if never edited).
  • 'Mark' column, which displays the value of the mark and the maximum value described by the marking category that the mark is under.
  • 'Comment' column, which displays a short comment written by the marker about the mark.
  • 'Action' column, which displays an 'Edit/Delete' button if you created the mark, or nothing if not. If you created the mark, click the 'Edit/Delete' button to edit or delete the mark.

The table may be sorted by clicking on the column headers. Clicking twice on the same header will reverse the sorting order. You can click on the following headers to sort the table:

  • 'Marker', which will sort the table by the marker's identifier.
  • 'Marking Category', which will group similar marking categories together (though in no particular order at the moment).
  • 'Timestamp', which will sort the table by the time the marks were last edited (or created if never edited).
  • 'Mark', which will sort the table by the left hand value of the mark - i.e., the result, not the maximum mark obtainable.
