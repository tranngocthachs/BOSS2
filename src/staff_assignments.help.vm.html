All staff pages

All staff pages have the same basic layout, which is described here.

Staff assignments page

The staff page shows you all of the assignments for a given module. An assignment is a subject that a student may submit solutions to.

Creating a new assignment

The actions menu on the left of the page has an item labelled 'Create new assignment...' which can be clicked to create a new assignment under the current module.

Switching mode

The assignments page has far too much information to display all at once, and is split into two modes. These modes are:

  • 'Submission', which will allow you to create/delete/edit assignments and their submissions, tests, and deadline revisions.
  • 'Marking', which will allow you to create/delete/edit assignments and their marking categories, markers, marking assignments and results.

The actions menu on the left of the page, below 'Create new assignment', has an entry to allow you to switch mode. If you are in Submission mode this will say 'Switch to marking view...' and clicking it will switch you to marking mode. Conversely, if you are Marking mode this will say 'Switch to submission view...' and clicking it will switch you to submission mode.

Submission mode

Assignments table

The main part of page contains a table describing the assignments in the current module. The table contains the following information for each assignment, one assignment to a row:

  • 'Assignment' column, which displays the name of the assignment.
  • 'Submissions' column, which displays the number of submissions made to the assignment. Next to this is a '(Manage...)' link which may be clicked to manage the submissions.
  • 'Tests' column, which displays the number of tests the assignment has. Next to this is a '(Manage...)' link which may be clicked to manage the tests.
  • 'Deadline Revisions' column, which displays the number of deadline revisions the assignment has. Next to this is a '(Manage...)' link which may be clicked to manage the deadline revisions.
  • 'Actions' column, which displays three buttons: 'Edit/Delete', 'Upload Resource', and 'Verify submission code'. These are explained in the next section.
You can click the headers of the columns to sort the tables. Clicking the same header twice will reverse the direction. The following headers may be used for sorting:
  • 'Submissions', which will sort by the number of submissions under the assignment.
  • 'Tests', which will sort by the number of tests under the assignment.
  • 'Deadline Revisions', which will sort by the number of deadline revisions under the assignment.

Actions

The 'Actions' column contains three buttons for each assignment. These are:

  • 'Edit/Delete', which allows you to edit or delete the assignment in question.
  • 'Upload Resource', which allows you to upload a zip file to the assignment that students may download (e.g. example, libraries, documentation, examples).
  • 'Verify submission code', which allows you to simulate a submission to generate a submission security code, in the event that BOSS2 loses a submission.

Marking mode

Assignments table

The main part of page contains a table describing the assignments in the current module. The table contains the following information for each assignment, one assignment to a row:

  • 'Assignment' column, which displays the name of the assignment.
  • 'Marking Categories' column, which displays the number of marking categories the assignment has. Next to this is a '(Manage...)' link which may be clicked to manage the marking categories.
  • 'Markers' column, which displays the number of markers the assignment has. Next to this is a '(Manage...)' link which may be clicked to manage the markers.
  • 'Marking Assignments' column, which displays the number of marking assignments the assignment has. Next to this is a '(Manage...)' link which may be clicked to manage the marking assignments.
  • 'Results' column, which displays the number of published results the assignment has. Next to this is a '(Manage...)' link which may be clicked to display and email out the published results.
  • 'Actions' column, which displays two buttons: 'Edit/Delete' and 'Publish Results'. These are explained in the next section.
You can click the headers of the columns to sort the tables. Clicking the same header twice will reverse the direction. The following headers may be used for sorting:
  • 'Marking Categories', which will sort by the number of marking categories under the assignment.
  • 'Markers', which will sort by the number of markers under the assignment.
  • 'Marking Assignments', which will sort by the number of marking assignments under the assignment.
  • 'Results', which will sort by the number of published results under the assignment.

Actions

The 'Actions' column contains two buttons for each assignment. These are:

  • 'Edit/Delete', which allows you to edit or delete the assignment in question.
  • 'Publish Results', which allows you to publish results (also re-publishing updated results).
