All student pages

All student pages have the same basic layout, which is described here.

Submission Successful page

The submission successful page notifies you that your submission was received by the BOSS2 system and notifies you of the unique security code for your submission.

Security code

The security code should be noted, and a copy of it will be emailed to you. If your submission is lost by the BOSS2 system, this code proves that you did manage to submit the given files.

Option to test your submission

Click the 'Test this submission' button to be taken straight to the the tests available for the submission that you just made. If you do not wish to test your submission, you may navigate away from the page using the breadcrumbs, menu, or browser back button.
