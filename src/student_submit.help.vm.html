All student pages

All student pages have the same basic layout, which is described here.

Submit page

The submit page allows you to submit a solution to an assignment. A solution is a set of required files for the markers to mark.

Submitting a solution

The main area of the page displays the following details about the submission that is to be made:

  • The Module that the submission is for. This can be clicked to take you to the Module Details page for that module.
  • The Assignment that the submission is for. This can be clicked to take you to the Assignment Details page for that Assignment.

Under this are a list of files that you are required to upload, followed by a button marked 'Submit'.

To perform your submission, you most browse your computer for each file requested before clicking the 'Submit' button.. If you do not wish to make a submission, you can navigate away from the page using the breadcrumbs, menu, or browser back button.

Note: there is no penalty for making multiple submissions. You may also specify later on which submission you prefer to be marked by navigating to the previous submissions page for that assignment.
