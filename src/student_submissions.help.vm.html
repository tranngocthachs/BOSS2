All student pages

All student pages have the same basic layout, which is described here.

Previous Submissions page

The Previous Submissions page provides a list of assignments that you have submitted solutions for in the past.

'Show..' menu

On the left hand of the page, under the navigation menu, is a menu that allows you to control which assignments are shown on this page. The items in the menu are:

  • 'Open assignments', which will display assignments you have submitted to that are still open.
  • 'Closed assignments', which will display assignments you have submitted to that have closed.
  • 'Open and closed assignments', which will display all assignments that you have submissions for.

Previous submissions table

The main part of the page displays a table listing a summary of all submissions you have made. The information displayed in the table is as follows:

  • 'Assignment' column, which will display the name of the assignment and the name of the parent module below it. Clicking on the assignment name will take you to the Assignment Details page for that assignment. Similarly, clicking on the module name will take you to the Module Details page for that module.
  • 'Number of Submissions' column, which will display the number of submissions you have made for the assignment in the 'Assignment' column.
  • 'Actions' column, which will provide a 'Test/Manage' button. Click this button to display, manage and/or test previous submissions for the assignment in the 'Assignment' column.

The table may be sorted by clicking on the headers. Clicking on the same header twice will reverse the order that the items are sorted by. You may click the following headers:

  • 'Number of Submissions', which will sort the table numerically by the number of submissions you have made for each assignment.
