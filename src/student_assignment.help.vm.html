All student pages

All student pages have the same basic layout, which is described here.

Assignment Details page

The Assignment Details page provides you details on an assignment that you can submit a solution for.

Assignment details

The main area of the page displays the following about the viewed assignment:

  • 'Assignment', which is the name of the assignment being viewed.
  • 'Parent module', which is the name of the module the assignment is for. This can be clicked to take you to the Module Details page for that module.
  • 'Due', which displays the time that you must submit your solution to the assignment by.
  • 'Markers', which displays a list of people that will be marking your assignment. You may click a person's name to write an email to them.
  • 'Resources', which provides a link to download any resources that the assignment administrator has provided, such as documentation or libraries.
  • 'Closing', which displays the time that solutions will not be accepted for the assignment after. Solutions submitted after the deadline but before the closing time may be marked at the module administrators' discretion.
