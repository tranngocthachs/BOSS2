All staff pages have the same basic layout, which is described here.

Upload assignment resource page

This page allows you to upload an assignment resource for students. An assignment resource could be anything that students may find useful, for example programming libraries, documentation, examples, source material, etc.

Uploading an assignment resource

To upload an assignment resource select the resource in the file choose under 'Resource file'.

Confirming

To confirm the upload, click the 'Perform Upload' button. This will overwrite any existing assignment resource that was uploaded. If you do not wish to upload an assignment resource, navigate away from the page using the menu, breadcrumbs or browser back button.
