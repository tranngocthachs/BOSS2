All staff pages

All staff pages have the same basic layout, which is described here.

Staff modules page

The staff page shows you all of the modules that you are able to administrate. A module is a collection of assignments, for example a subject on a course.

Modules table

The main part of page contains a table describing the modules that you are allowed to administrate. The table contains the following information for each module, one module to a row:
  • 'Module' column, which displays the unique identifier of the module and below it its name.
  • 'Assignments' column, which displays the number of assignments under the module, as well as two links to manage them. These links are explained in the next section.
  • 'Students' column, which displays the number of students for the module as well as a '(Manage...)' link. To manage the students for this module, click the '(Manage...)' link.
You can click the headers of the columns to sort the tables. Clicking the same header twice will reverse the direction. The following headers may be used for sorting:
  • 'Assignments', which will sort by the number of assignments under the module.
  • 'Students', which will sort by the number of students under the module.

Accessing assignments

Under the number of assignments are two links that you can click to manage assignments. These are:
  • '(Manage submission...)', which will allow you to create/delete/edit assignments and their submissions, tests, and deadline revisions.
  • '(Manage marking...)', which will allow you to create/delete/edit assignments and their marking categories, markers, marking assignments and results.
