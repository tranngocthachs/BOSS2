All staff pages

All staff pages have the same basic layout, which is described here.

Staff module students page

The module students page shows you all of the students for a given module. A student for a module is allowed to access the module for submission if registration is required.

Add a new student

To add an student, click the 'Add student' item in the 'Actions' menu on the left hand side of the page.

Students table

The main part of page contains a table describing the student in the current module. The table contains the following information for each student, one student to a row:
  • 'Person' column, which displays the identifier for the person and below it the person's chosen name.
  • 'Email' column, which displays the e-mail address of the student. You may click the address to compose an email to the student.
  • 'Actions' column, which displays a button marked 'Remove student'. Click this button to remove the student from the module.
You can click the headers of the columns to sort the tables. Clicking the same header twice will reverse the direction. The following headers may be used for sorting:
  • 'Person', which will sort by the student's unique identifier.
  • 'E-mail Address', which will sort by the student's email address.
