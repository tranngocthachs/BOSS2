All admin pages

All admin pages have the same basic layout, which is described below.

Title, breadcrumbs and greeting

Under the logo is the title of the current page. Below this is a set of 'breadcrumbs', showing the navigational hierachy of the current page. You may click any page in the navigational hiearchy to return to that page.

On the top right is a greeting showing your name and providing a 'Log out' link that will log out you out.

Admin navigation menu

On the left hand of the page is a menu providing links to navigate the admin section of the BOSS2 system. The items in the menu are:

  • 'Manage curriculum', which will take you to a page listing the models in the database. A model is a collection of modules, for example an academic year.
  • 'Manage people', which will take you to a page listing the people in the database.
  • 'Log out', which will log you out of the BOSS2 system.

Help

On the bottom right of the page there are three links:

  • Help, to take you to the online help.
  • About BOSS, to take you to the credits for the BOSS2 system.
  • Accessibility, to take you to a page describing the accessibility features of the BOSS2 system.
