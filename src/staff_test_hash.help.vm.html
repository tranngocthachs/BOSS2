All staff pages

All staff pages have the same basic layout, which is described here.

Test hash page

The test hash page allows you to simulate submitting a solution to an assignment to generate a verification code. This is useful if, for example, BOSS2 loses a submission and a student has a submission verification code to prove they made the submission.

Hashing a solution

There is a list of files that you are required to upload, followed by a button marked 'Hash'.

To perform the hash, you most browse your computer for each file requested before clicking the 'Hash' button.. If you do not wish to make a hash, you can navigate away from the page using the breadcrumbs, menu, or browser back button.
