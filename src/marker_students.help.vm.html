All marker pages

All marker pages have the same basic layout, which is described here.

Students page

The students page displays a list of students that you can mark (or moderate, if you are a moderator) for an assignment, and provides access to their submissions and marks.

Students table

The main part of the page contains a table displaying information on the students that you may mark. The table provides the following information:

  • 'Person' column, which will display the name and email address of the student to be marked, or '(Blind marking)' if that student is to be marked anonymously. If the email address is displayed, clicking on it will allow you to email the student.
  • 'Submissions' column, which will display the number of submissions that the student has made for the assignment.
  • 'Last marked by you' column, which will display the date that you last created a mark for this student (if you have created one).
  • 'Action' column, which contains an 'Access submissions' button and a 'Mark/moderate Student' button, which explained in the next sections.

The table may be sorted by clicking on the column headers. Clicking twice on the same header will reverse the sorting order. You can click on the following headers to sort the table:

  • 'Person', which will sort the table by the student's identifier.
  • 'Submissions', which will sort the table by the number of submissions that the student has made.
  • 'Last marked by you', which will sort the table by the time that you last marked the student.

Accessing submissions

To access a student's submissions in order to download them or test them, click the 'Access submissions' button in the 'Actions' column of the desired student's row.

Marking a student

In order to access the marks for the student (in order to create, edit, or delete them), click the 'Mark/moderate Student' button in the 'Actions' column of the desired student's row.
