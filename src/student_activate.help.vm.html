All student pages

All student pages have the same basic layout, which is described here.

Activate Submission page

The Activate Submission page lets you confirm that you want a submission to be the one that is marked by a marker.

Activating a submission

The main area of the page displays the following details about the submission that is to be activated:

  • The Module that the submission was submitted to. This can be clicked to take you to the Module Details page for that module.
  • The Assignment that the submission was submitted to. This can be clicked to take you tot he Assignment Details page for that Assignment.
  • The time that the submission was performed.

Under this there is a check box to confirm that you wish to activate the submission, and a button marked 'Activate Submission'. To activate your submission, check the box to confirm and click 'Activate Submission'. If you do not wish to activate your submission, you can navigate away from the page using the breadcrumbs, menu, or browser back button.
