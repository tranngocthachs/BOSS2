All staff pages

All staff pages have the same basic layout, which is described here.

Staff tests page

The tests page shows you all of the tests for a given assignment. Tests are automated operations students and/or markers can run to verify the correctness of their solutions.

Creating a new test

To create a new test, click 'Create new test...' under the Actions menu on the left hand side of the page.

Tests table

The main part of page contains a table describing the tests in the current assignment. The table contains the following information for each test, one test to a row:

  • 'Test' column, which displays the name of the test.
  • 'Student accessible?' column, which displays whether or not students are allowed to run the test.
  • 'Actions' column, which displays two buttons: 'Edit/Delete Test' and 'Upload Test Resource'. These are explained in the next section.

You can click the headers of the columns to sort the tables. Clicking the same header twice will reverse the direction. The following headers may be used for sorting:

  • 'Student accesible?', which will group tests by whether students may run them or not.

Actions

Each test has the following actions available. Click the button to perform the action:

  • 'Edit/Delete Test', which allows you to modify or delete the test (and test parameters).
  • 'Upload Test Resource', which allows you to upload a resource file for testing purposes.
