All staff pages

All staff pages have the same basic layout, which is described here.

Edit Assignment page

This page allows you to edit, delete or create an assignment.

Creating/editing the assignment

An assignment has the following values that must be filled in:

  • 'Name': a custom name for the assignment. Names do not need to be unique.
  • 'Deadline': the date and time at which students are notified they are overdue for submission.
  • 'Opening date': the date and time at which students are granted access to the assignment.
  • 'Closing date': the date and time at which students are denied access to the assignment, unless they have deadline revisions granting them extra time.
  • 'Allow delete': a flag stating whether students are allowed to delete previous submissions. Not recommended as it destroys a potential 'paper' trail.

Deleting the assignment

If you are not creating an assignment, under the values for the module there will be a checkbox marked 'Delete this assignment'. To delete the assignment rather than edit it, select this checkbox. If you are creating an assignment this checkbox will not be visible.

NOTE: If the assignment has anything that depends on it (e.g. submissions, tests, marking categories, results) you will not be able to delete it.

Confirming

If you are editing or deleting an assignment there will be a button labeled 'Perform Edit/Delete'. If you are creating an assignment the button will be labeled 'Perform Create'. Once you are sure of the values you have selected, click this button to commit the details to the BOSS2 database. If you do not wish to create, edit or delete an assignment navigate away from the page using the navigation menu, breadcrumbs or browser back button.
