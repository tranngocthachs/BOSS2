Test Results page

The test results page either notifies you that testing is in progress or the test results for any tests that you have run previously.

Title

Under the logo is the title of the current page.

Close window

On the left hand of the page is a menu providing a single 'Close window' link. Clicking this will close the window if you have Javascript available and enabled in your browser.

Test results

If tests are in progress, the page will display 'Testing in progress, please wait.' Once the tests are complete, the main area of the page displays the following details about the test results in a table:

  • 'Result' column, which displays the numerical mark that the test produced.
  • 'Output' column, which displays the output from running the test.
  • Below these columns is a single line comment summary.
