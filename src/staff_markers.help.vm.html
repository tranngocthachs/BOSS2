All staff pages

All staff pages have the same basic layout, which is described here.

Staff assignment markers page

The assignment markers page shows you all of the markers for a given assignment. A marker for an assignment is able to be given marking assignments to ultimately mark a student's submission.

Add a new marker

To add an marker, click the 'Add marker' item in the 'Actions' menu on the left hand side of the page.

Markers table

The main part of page contains a table describing the marker in the current assignment. The table contains the following information for each marker, one marker to a row:
  • 'Person' column, which displays the identifier for the person and below it the person's chosen name.
  • 'Email' column, which displays the e-mail address of the marker. You may click the address to compose an email to the marker.
  • 'Actions' column, which displays a button marked 'Remove marker'. Click this button to remove the ,marker from the module.
You can click the headers of the columns to sort the tables. Clicking the same header twice will reverse the direction. The following headers may be used for sorting:
  • 'Person', which will sort by the marker's unique identifier.
  • 'E-mail Address', which will sort by the marker's email address.
